The Zeebe Node client is available via [npm](https://www.npmjs.com/package/zeebe-node).

## Add to an existing project

{{< tabs >}}
{{< tab Bash >}}
{{< highlight bash >}}
npm i zeebe-node
{{< /highlight >}}
{{< /tab >}}

{{< /tabs >}}

## Create a new Node.js project

{{< tabs >}}
{{< tab Bash >}}
{{< highlight bash >}}
mkdir myproject
cd myproject
npm init -y
npm i zeebe-node
{{< /highlight >}}
{{< /tab >}}
{{< /tabs >}}
